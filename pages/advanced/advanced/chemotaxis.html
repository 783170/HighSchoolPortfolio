If you click any of the four screens below one of my animal designs will be drawn. Each of these animals has their own class that extends the SeaAnimal class so they can inherit the same move function. I create each of these animals from scratch using ellipsis, curves, lines, triangles, and other basic shapes.

- the fish are draw randomly either orange or pink.
- the shark is the most basic and is drawn the same way each time
- the seahorse has an overridden move method that uses a sine function to create an up and down movement
- the puffer fish uses its original x and y position and the distance it has travelled to grow. As it travels from left to right the fish grows taller and its spines grow propotionally to give it a more authentic puffer fish look
