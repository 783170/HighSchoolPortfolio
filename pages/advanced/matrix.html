Advanced Computer Science

Matrix

  • Return To Home

This was my final project for linear algebra. I worked with my classmates Hope Wanken and Maggie McNeary. We worked to create a matrix calculator. Our program was able to take in a number of rows and columns and then inputs for each of the matrix values. If a place is left blank than it automatically becomes a zero. Then each of our buttons performs a different type of calculation on the original matrix (I presented on the Null Space).
