This code worksby creating a rectangle that follows the mouse position using mouseX and mouseY, and it uses if statements to keep the ball on the page. The ball moves by create an cicle with a diameter that moves by a certain value every fram. These values are defined by the variable temp. As the ball makes contact with the right side, left side, top side, or the rectangle the values xTemp and yTemp change polarity in order to make the ball bounce. There is also an if statement that checks when the ball and the rectangle make contact, this increases the temp value and makes the ball speed up. One issue I had was the ball leaving a trail behind itself. To fix this I included the functions that created my background in my draw method so that the previous positions get covered up.
